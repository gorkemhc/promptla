# yapay-zeka
Projenin amacı, kullanıcıların yapay zekâ uygulamalarını doğru ve etkili bir şekilde kullanabilmesini sağlamaktır. Kullanıcıların yanlış prompt girmesini engelleyecek, yapay zekâ çeşitlerini ve kullanım yöntemlerini açıklayan, aynı zamanda hazır prompt örnekleri sunan bir web sayfası geliştirmek hedeflenmektedir.
