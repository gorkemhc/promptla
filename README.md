## Proje Hakkında

**PROMPTLA**, kullanıcıların yapay zekâ araçlarını daha bilinçli ve etkili kullanmasını amaçlayan modern bir öğrenme platformudur.

Proje; yapay zekâ model rehberleri, kategori bazlı prompt örnekleri, canlı önizlemeli Prompt Builder, sosyal paylaşım alanları ve HTML Canvas tabanlı mini oyunlar içerir. Statik HTML, CSS ve JavaScript yapısı sayesinde herhangi bir sunucu kurulumu olmadan çalıştırılabilir ve GitHub Pages üzerinde ücretsiz yayınlanabilir.

## Öne Çıkan Özellikler

| Özellik | Açıklama |
|---|---|
| 🤖 **Yapay Zekâ Model Rehberleri** | Popüler yapay zekâ araçlarını ve kullanım alanlarını tanıtan sayfalar |
| ✍️ **Prompt Builder** | Kullanıcı girdilerine göre anlık olarak güncellenen prompt oluşturucu |
| 📚 **Prompt Kütüphanesi** | Kod, görsel, video, eğitim ve içerik üretimi için hazır prompt örnekleri |
| 🎮 **Mini Oyunlar** | HTML Canvas ve JavaScript ile hazırlanmış oynanabilir örnekler |
| 🌐 **Responsive Arayüz** | Masaüstü, tablet ve mobil cihazlara uyumlu tasarım |
| ⚡ **Statik Çalışma** | Backend veya paket kurulumu gerektirmeden doğrudan çalıştırılabilir |
| 🔎 **SEO Temeli** | `robots.txt`, `sitemap.xml`, meta etiketleri ve özel 404 sayfası |
| 🌙 **Modern Tasarım** | Koyu tema, neon mavi vurgular ve akıcı etkileşimler |

## Yerel Olarak Çalıştırma

Projeyi bilgisayarınıza klonlayın:

```bash
git clone https://github.com/gorkemhc/promptla.git
```

Proje klasörüne girin:

```bash
cd promptla
```

Ardından `index.html` dosyasını tarayıcıda açın.

Windows PowerShell üzerinden açmak için:

```powershell
Start-Process .\index.html
```

Daha rahat geliştirme için Visual Studio Code üzerindeki **Live Server** eklentisi kullanılabilir.

## Canlı Demo

GitHub Pages yayını:

**https://gorkemhc.github.io/promptla/**

GitHub Pages henüz aktif değilse:

1. Repo içindeki **Settings** bölümüne girin.
2. Sol menüden **Pages** seçeneğini açın.
3. Kaynak olarak **Deploy from a branch** seçin.
4. Branch olarak `main`, klasör olarak `/root` belirleyin.
5. **Save** butonuna basın.

## Mevcut Sürümün Kapsamı

Bu sürüm statik olarak çalışır. Giriş, kayıt, yorum, sosyal paylaşım ve yönetim paneli gibi bazı alanlar demo amaçlıdır veya tarayıcı tarafında çalışır.

Gerçek kullanıcı doğrulaması, kalıcı veritabanı ve sunucu taraflı yönetim işlemleri için ilerleyen sürümlerde Firebase veya farklı bir backend altyapısı entegre edilebilir.
