Claude Code nedir?

Claude Code, terminal ve desteklenen editörler üzerinden proje klasörünle çalışan bir kodlama aracıdır. Dosya yapısını okuyabilir, plan hazırlayabilir, kod düzenleyebilir ve çalıştırmasına izin verdiğin komutlarla doğrulama yapabilir.

CLAUDE.md proje talimatı

CLAUDE.md, proje boyunca korunmasını istediğin kuralları açık biçimde yazdığın talimat dosyasıdır. Tasarım renkleri, klasör yapısı, test komutları ve değiştirilmemesi gereken alanlar burada belirtilebilir.

CLAUDE.md örneği
# PROMPTLA Proje Kuralları
- Mevcut koyu lacivert, turkuaz ve mor renk sistemini koru.
- PHP, Firebase veya veritabanı ekleme.
- Tüm sayfalar index.html çift tıklanınca çalışabilsin.
- Bir dosyayı değiştirmeden önce mevcut davranışı analiz et.
- Mobil menüyü ve erişilebilirliği bozma.
- Değişiklik sonunda git diff özetini ve test sonuçlarını yaz.
